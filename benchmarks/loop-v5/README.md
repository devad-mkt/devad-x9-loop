# X9 Loop v5 Benchmarks

This folder extends the retained model-routing ledger with orchestration tests.

Measure end-to-end cost, not one-turn cost:

- tokens and wall time per accepted completion;
- first-pass proof rate;
- orchestration retries;
- role/delivery identity errors;
- lost local-work incidents;
- resource conflicts;
- Thinx Ultra return to xhigh.

Deterministic identity tests run on every package validation. Real model tests
must append dated evidence and never replace older results.
