# Supporting Skills

The public kit ships an intentional v6 set. It does not overwrite an installed
copy until the installer has created a local backup and validated the candidate.

| Skill | Public purpose |
| --- | --- |
| `codex-x9-backup` | Configurable private backup, redaction, scan, and restore proof |
| `codex-token-budget` | Task/model/callback and loop cost diagnosis and benchmark routing |
| `devad-memory` | Historical retrieval; never active routing authority |

Private live-skill archives, local benchmark runs, and machine-specific backup
remotes are intentionally not distributed. The public backup skill requires a
user-supplied private clone and optional remote.
